## 1. 标签定义（最终确定版）

训练标签统一为：

    y = 1{close[t0+5m] > open[t0]}

其中：

- t0 ∈ {..., 12:00, 12:05, 12:10, ...}
- 使用 1m 数据构造
- open[t0] ≈ 窗口起点价格（12:00:00）
- close[t0+5m] ≈ 窗口结束价格（12:04:59）

### 为什么用这个定义

相比其他写法：

❌ close[t0] → 错位（12:00:59）  
❌ 任意时间滚动 → 与真实合约不对齐  

✅ open[t0] → 最接近真实合约起点  
✅ grid 对齐 → 和 Polymarket 完全一致的时间结构
